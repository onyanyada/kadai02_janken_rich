## ⑤工夫した点・こだわった点

- ガチャで当選したアイテムを活用できる(鍵：隠された好感度を見られる、猫：好感度を上げる、茶葉：選択肢が増える)
- 各ストーリーで選ぶ選択肢により、攻略対象の好感度が変化し、結末が変わる（最後に攻略対象を選ぶ）
- 各ストーリーの順番がランダムに変化
- ランダムでイベント(通常以外のストーリー以外)が発生
- ハードモードを適用できる
